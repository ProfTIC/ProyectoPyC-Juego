1ª fase. Elección de las herramientas de trabajo del proyecto

Herramientas de trabajo

Vamos a usar las siguientes herramientas:

  • Framework de greenfoot.
  • Sistema de control de versiones de github. En el anexo se explica el procedimiento para crear un repositorio, añadir colaboradores y subir archivos

Tarea 1

  • Instalar el framework de greenfoot
  • Cada miembro del equipo se creará una cuenta en github
  • Uno de los componentes del equipo creará un proyecto y añadirá como colaboradores al resto de compañeros

Qué se entrega en esta fase

  • Enlace al repositorio github
  • Solo lo tiene que entregar uno de los componentes del equipo
